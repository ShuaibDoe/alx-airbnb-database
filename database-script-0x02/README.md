##  Overview
This script populates the **Airbnb Clone database** with realistic, sample data representing users, properties, bookings, payments, and reviews.  
It is designed to simulate real-world application usage and support testing for queries, analytics, and backend API development.

---

##  Objective
- Populate the Airbnb database with consistent, normalized data.  
- Reflect realistic relationships between hosts, guests, and properties.  
- Ensure each core entity (User, Property, Booking, etc.) has valid relational links.
